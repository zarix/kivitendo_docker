# Kivitendo Docker

## Version

* 3.7.0

## Installation

### Build

```
git clone https://github.com/zarix/kivitendo
cp .env.template .env
docker-compose build
```

### Start
```
docker compose up -d
```
Point your browser to http://localhost:8080 and login using the default username and password:

    - username: __admin__
    - password: __admin__
